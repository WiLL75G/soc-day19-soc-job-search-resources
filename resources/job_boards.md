## Job Alert Setup

Set up daily job alerts on these platforms:

**LinkedIn:**
- Go to Jobs → Search → "SOC Analyst"
- Click "Set Alert" Daily frequency
- Add your location + Remote option

**Indeed:**
- Search "SOC Analyst"
- Click "Get email alerts"
- Set to daily

**Google Alerts:**
- Go to alerts.google.com
- Create alert for: `"SOC Analyst" "hiring" OR "now hiring"`

## How to Apply Strategically

```
RULE 1: Apply within 24 hours of posting
         New postings get 3x more views from recruiters

RULE 2: Tailor your resume for each application
         Mirror the exact keywords from the job description

RULE 3: Follow up after 5 business days
         A LinkedIn message to the hiring manager sets you apart

RULE 4: Apply to 5-10 roles per day minimum
         Volume matters at the entry level

RULE 5: Track every application
         Use a spreadsheet company, role, date, status
```
